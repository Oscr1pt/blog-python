# Bienvenido a Oh My Python!

Esta documentación ha sido creada como una guía estructurada para acompañarte en el proceso de aprendizaje de Python desde cero.

El objetivo principal es ofrecer explicaciones claras, ejemplos prácticos y una progresión lógica que te permita comprender cada concepto paso a paso.

---

## ¿Qué encontrarás en esta guía?

A lo largo de este manual trabajaremos los fundamentos esenciales del lenguaje, organizados de manera progresiva:

- Variables y tipos de datos
- Operadores
- Estructuras condicionales
- Bucles (loops)
- Funciones
- Estructuras de datos
- Introducción a la programación orientada a objetos

Cada sección incluirá ejemplos prácticos y explicaciones diseñadas para reforzar la comprensión.

---

## ¿Cómo aprovechar esta documentación?

Para obtener el máximo beneficio:

- Sigue el orden propuesto en las secciones.
- Escribe y ejecuta cada ejemplo por tu cuenta.
- Experimenta modificando el código.
- No avances sin comprender el tema anterior.

La programación se aprende practicando.

---

## Recomendación antes de comenzar

Asegúrate de tener Python instalado en tu computadora y un editor de código listo para usar.

Si aún no lo has hecho, revisa la sección de [primeros pasos](primerosPasos) antes de continuar.

---

## Enfoque del curso

Este manual está diseñado para enfocarse en la comprensión lógica antes que en la memorización.

Más importante que recordar la sintaxis, es entender cómo resolver problemas utilizando programación.
